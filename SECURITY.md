# Security Policy

## Supported Versions
Only the latest release version of Capsule Infinity receives active security patches.

## Reporting a Vulnerability
We take the security of Capsule Infinity seriously. If you find a vulnerability, please report it privately.
* Do **not** open a public issue.
* Email security reports directly to: `capsuleinfinity.support@gmail.com` (or open a private security advisory on GitHub).
We will investigate and patch verified vulnerabilities within 14 days of receipt.
