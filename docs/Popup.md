# Popup Window Component

The Popup (`popup/`) is the lightweight interface displayed when you click the extension icon in your browser toolbar.

![Popup Explanation](../assets/diagrams/popup_explanation.svg)

## Key Features
* **Stats Panel Grid**: Displays real-time counts of saved capsules, folders, and teams.
* **Quick Actions**: Includes shortcuts to Capture the active tab page, create manual text capsules, or launch the sidebar.
* **Recent List**: Displays the last 6 saved items, with quick buttons to copy payload contents or delete files.
* **Creator Footer**: Elegant copyright badge aligned at the absolute bottom of the panel.
