# Product Roadmap

See what features are planned for future releases of Capsule Infinity.

![Roadmap timeline](../assets/diagrams/roadmap.svg)

## Milestone 1: Core Performance (Completed)
* [x] Injected scraping scripts for ChatGPT and Claude.
* [x] Chunked messaging buffer loop to save massive conversations.
* [x] 8-second timeout guard to prevent UI freezes.

## Milestone 2: Cloud Sync (Completed)
* [x] Supabase database sync client setup.
* [x] Google account OAuth session validation.
* [x] Offline local cache fallback database.

## Milestone 3: Collaborative Workspace (In Progress)
* [ ] Shared team capsules list.
* [ ] Dynamic team code invitations and member management.

## Milestone 4: Offline-First Queue (Future)
* [ ] Persistent queue manager to batch operations and sync changes when network comes online.
* [ ] Mobile browser extension wrapper.
